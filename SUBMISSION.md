# CryptoPulse - Slack Agent Builder Challenge Submission

## What
CryptoPulse brings real-time crypto trading signals into Slack. 
Traders get prices, signals, and market reports without leaving their workspace.

## How
- Slack Bolt (Python) + Socket Mode
- Binance API for real-time prices
- RSI/MACD technical analysis for signals
- MCP-ready architecture

## Demo
@CryptoPulse price btc → returns live BTC price
@CryptoPulse signal → returns multi-timeframe trading signals
@CryptoPulse report → returns daily market summary

## Tech
- Python + Slack Bolt
- Socket Mode (no server needed)
- Binance real-time WebSocket
- Multi-timeframe RSI/MACD/Bollinger analysis

## Try it
Invite @CryptoPulse to your Slack workspace

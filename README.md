# CryptoPulse - Slack-native Crypto Trading Agent

## What it does
Real-time crypto market assistant inside Slack. Get prices, trading signals, and market reports without leaving your workspace.

## Commands
- `price btc` - Real-time crypto prices
- `signal` - RSI/MACD multi-timeframe trading signals
- `report` - Daily market summary
- `price eth/sol/bnb` - Any supported coin

## Tech Stack
- Slack Bolt (Python)
- Binance API (real-time prices)
- Socket Mode (no server needed)
- Multi-timeframe technical analysis

## Why Slack?
Traders live in Slack. Bringing crypto signals natively into the workspace eliminates context switching between trading platforms and team communication.
